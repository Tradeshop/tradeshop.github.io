TradeShop Documentation

Official Addons

Addons are Bukkit/Spigot plugins which hook into the TradeShop Addon Framework (TSAF), which is made of a set of custom event classes, so that the TradeShop plugin may fire custom events when a Trade-related action takes place.
This way, we can offer more plugins which rely on the TSAF, and offer more spectacular features.

You can also find inofficial addons made by other people. On here, though, we will provide a list of official addons, which we provide support and updates for. For other TradeShop-related projects, feel free to checkout our development hub, aka. our GitHub organization here.

For now, no official addons have been released yet. This page will be updated when we have something more to offer.
